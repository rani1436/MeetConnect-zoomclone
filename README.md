# Zoom Clone 🎥

A real-time video conferencing application inspired by Zoom, enabling users to create and join virtual meetings seamlessly. The platform supports video/audio communication, chat, and screen sharing for an interactive collaboration experience.

## ✨ Features

* 📹 Real-time video conferencing
* 🎤 Mute/Unmute microphone
* 📷 Turn camera on/off
* 💬 Real-time chat
* 🖥️ Screen sharing
* 🔗 Create and join meeting rooms
* 📱 Responsive design

## 🛠️ Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Real-Time Communication

* WebRTC
* Socket.io

### Database

* MongoDB

🚀 Getting Started

 Prerequisites

* Node.js
* npm

 Installation

1. Clone the repository

```bash
git clone https://github.com/rani1436/zoom-clone.git
```

2. Navigate to the project directory

```bash
cd zoom-clone
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm start
```
